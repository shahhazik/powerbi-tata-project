# Tata Sales Analysis Dashboard | Power BI

This repository contains a Power BI dashboard built to analyze Tata's sales performance across multiple regions and time periods.  
The project focuses on visualizing key business metrics, growth trends, and regional insights.

---

##  Project Overview
This dashboard helps stakeholders understand:
-  **Overall Sales Performance**
-  **Region-wise Revenue Distribution**
-  **Top-performing Products**
-  **Year-over-Year Growth Trends**
-  **Key Performance Indicators (KPIs)**
